## Projects

### 1. Underwater Robotics – Sonar-Based Autonomous Navigation

Autonomous navigation system for an unmanned underwater vehicle (UUV) using sonar-based perception.

- Designed real-time pipelines for sensor buffering and decision-making  
- Applied median and moving average filtering to stabilize noisy sonar data  
- Developed state-based control logic for autonomous navigation  

Award: Best Software Award – TEKNOFEST Unmanned Underwater Vehicles Competition  

Includes:
- System architecture diagrams  
- Workflow documentation  
- Demonstration media  

[View Project](./1_Underwater_Robotics_Sonar)

### 2. Brain-Computer Interface – EEG Signal Classification

Machine learning system that translates EEG signals into structured command outputs.

- Built full signal processing pipeline from acquisition to classification  
- Applied preprocessing and feature extraction techniques  
- Implemented Support Vector Machine (SVM) classifier  

Includes:
- Signal processing workflow diagrams  
- Classification pipeline documentation  
- Visualization of EEG signals  

[View Project](./2_BCI_Signal_Classification)

### 3. Aircraft Surface Defect Detection – Computer Vision

Deep learning-based system for detecting structural defects on aircraft surfaces.

- Developed CNN-based model for defect detection  
- Applied image preprocessing and data augmentation techniques  
- Built a demo for visualizing predictions  

Includes:
- Full source code
